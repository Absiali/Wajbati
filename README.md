# 🍽️ وجبتي

تطبيق ويب لإدارة طلب الوجبات في المستشفيات

## المميزات

### 👨‍⚕️ واجهة المريض
- اختيار وجبات الفطور والغداء والعشاء
- اختيار مشروب وفاكهة لكل وجبة
- طلب الوجبات للغد
- واجهة سهلة وودية

### 🏥 لوحة الإدارة
- إدارة الوجبات (إضافة/حذف/إظهار/إخفاء)
- إدارة المشروبات (إضافة/حذف/إظهار/إخفاء)
- إدارة الفواكه (إضافة/حذف/إظهار/إخفاء)
- عرض جميع الطلبات المسجلة

## التقنيات المستخدمة

- **HTML5** للهيكل
- **CSS3** للتصميم
- **JavaScript** للوظائف
- **Bootstrap 5** للمظهر المتجاوب
- **LocalStorage** لتخزين البيانات

## البدء السريع

1. افتح ملف `index.html` في المتصفح
2. ستجد واجهة المريض مباشرة
3. انقر على زر "لوحة الإدارة" للدخول إلى إدارة الوجبات

## كيفية الاستخدام

### للمريض:
1. اختر التاريخ (افتراضياً يكون غداً)
2. اختر الوجبات والمشروبات والفواكه
3. انقر على "تأكيد الطلب"

### للإدارة:
1. انقر على "لوحة الإدارة"
2. يمكنك:
   - **إضافة عنصر**: أدخل الاسم وانقر إضافة
   - **إظهار/إخفاء**: انقر على الزر للتحكم في التوفر
   - **حذف**: انقر على 🗑️ لحذف العنصر
   - **عرض الطلبات**: انقر على "الطلبات" لرؤية جميع الطلبات

## البيانات المحفوظة

تُحفظ جميع البيانات محلياً في متصفحك:
- `wajbati_meals`: الوجبات
- `wajbati_drinks`: المشروبات
- `wajbati_fruits`: الفواكه
- `wajbati_orders`: الطلبات
